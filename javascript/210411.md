# 함수

## 익명함수와 선언적 함수의 차이 

1. 익명함수
```javascript
const f = function (매개변수, 매개변수) {
    return 리턴값
}
```

2. 선언적 함수
```javascript
function f (매개변수, 매개변수) {
    return 리턴값
}
```

익명함수는 우리가 코드를 읽을 때와 같은 순서로(순차적) 함수가 선언되지만, 선언적 함수는 우리가 코드를 읽는 순서와 다른 순서로 함수가 선언됨. 그래서 안전하게 사용할 수 있는 익명함수를 더 선호.
